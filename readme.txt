WP Yogi Simple theme
====================

This theme was created as part of a tutorial provided at WPyogi.com.

http://www.wpyogi.com/2011/04/build-a-simple-custom-theme/

This theme is the first one is a series.

You can check out a demo of the theme here.
http://wpyogi.com/simple/
 


----------
INSTALLATION

To install upload wpyogi-simple to your /wp-content/themes directory. 
Import the xml file provided
